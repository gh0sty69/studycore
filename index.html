Offline Sync Guide

Some school or corporate Wi-Fi firewalls strictly block the Firebase Cloud Connection. If your app says "No Cloud Configured" or fails to sync data automatically, use this manual USB method to bypass the firewall:

  1. On your Home PC, go to Settings -> Data Management, and click exactly Download JSON.
  2. Save that .json file to a USB flash drive (or email it to yourself).
  3. Go to your School PC. Create a brand new account (any name works).
  4. Open Settings -> Data Management, and click exactly Upload JSON.
  5. Select the file from your USB drive. Your Home PC data will instantly overwrite the dummy account, fully offline!
